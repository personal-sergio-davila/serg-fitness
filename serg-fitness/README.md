# Serg Fitness App

Phase 1 — Meals, Grocery, Workout Logger

## Stack
- React + Vite
- Supabase (Postgres database)
- Vercel (hosting)

## Deploy steps

1. Create a GitHub repo called `serg-fitness`
2. Upload all these files to it
3. Go to vercel.com → New Project → Import that repo
4. Vercel auto-detects Vite — just hit Deploy
5. Done. Your app is live at a `.vercel.app` URL

## Local dev
```
npm install
npm run dev
```
